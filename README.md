# Amazon Clone (HTML & CSS)

A simple Amazon-inspired e-commerce user interface built using HTML and CSS. This project focuses on layout structure, responsiveness, and clean UI design.

## 🚀 Features

* Responsive design (desktop-focused)
* Navigation bar with search section
* Hero banner section
* Product grid layout
* Clean and structured UI

## 🛠️ Tech Stack

* HTML
* CSS
